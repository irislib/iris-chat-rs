# Iris Chat 2026.5.10

- Fixes iOS message bubbles rendering centered instead of left/right-aligned by sender.
- Restores reliable chat scrolling on iOS; the swipe-to-reply gesture was conflicting with vertical scroll. Long-press a message for Reply, Info, Copy, or Delete.
- Closing the macOS window now hides Iris Chat instead of orphaning the dock icon — clicking the dock brings the window straight back.
- Minimizing on macOS folds into the existing dock icon instead of leaving a screenshot thumbnail next to the trash.
- Settings hides the "Install update" button when there's nothing to install.
